Befor running:
- all input/output files must be in the cloudAnalyst folder for the make run command to execute successfully.

Instructions for running:

use command 'make run' in cloudAnalyst folder to run.
You will be prompted for three pieces of information in this order:
1. the name of the input file. Enter to continue
2. the name of the output file. Enter to continue
3. A code, either 0 or 1.

code: (USE CODE 0)
0 - run program once with given files
1 - runs program several times with different cutoffs and files (for testing purposes)
